# Sockets en C

* Para compilar todos los archivos  
```make all```
* Para compilar el servidor  
```make server```
* Para compilar el cliente  
```make client```

* Para eliminar los ejecutables  
```make clean```

* Para ejecutar el servidor  
```./build/server [puerto]```

* Para ejecutar el cliente  
```./build/client [host] [puerto]```
